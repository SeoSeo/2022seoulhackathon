## 주제 및 응모 조건
- 서울시 IoT 공공도시데이터 중 1개 이상의 데이터 활용 필수

## 응모 분야
- 3개 분야 중 1개 선택 (주제: 제한 없음)
  - ① 웹/앱 서비스 제작 ② 분석보고서 ③ 인포그래픽 제작
-> 분석보고서 로서 주자창이어드림 팀으로서 참가.
